## Common Issues & Solutions

### Issue: Cannot log in

**Troubleshooting:**
1. Check email/password
2. Try password reset
3. Clear browser cache
4. Check account status

### Issue: Leave request not showing

**Troubleshooting:**
1. Refresh page
2. Check date filters
3. Verify permissions
4. Check browser console for errors

### Issue: Attendance not recording

**Troubleshooting:**
1. Check system time
2. Verify employee status
3. Check attendance settings
4. Review attendance logs
