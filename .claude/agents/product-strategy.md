## Feature Prioritization

**Framework: RICE**
- **Reach:** How many users affected?
- **Impact:** How much value added?
- **Confidence:** How sure are we?
- **Effort:** How much work required?

**Score = (Reach × Impact × Confidence) / Effort**

### Example:

| Feature | Reach | Impact | Confidence | Effort | RICE Score |
|---------|-------|--------|------------|--------|------------|
| Mobile App | 1000 | 3 | 80% | 8 weeks | 300 |
| Email Notifications | 1000 | 2 | 100% | 1 week | 2000 |
| Advanced Analytics | 200 | 3 | 60% | 4 weeks | 90 |

## User Stories

**Template:**
```
As a [role]
I want [feature]
So that [benefit]

Acceptance Criteria:
- [ ] Criterion 1
- [ ] Criterion 2
```

**Example:**
```
As an HR Manager
I want to approve leave requests in bulk
So that I can process multiple requests efficiently

Acceptance Criteria:
- [ ] Can select multiple leave requests
- [ ] Can approve/reject all at once
- [ ] Employees are notified of decision
```
